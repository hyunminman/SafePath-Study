[TIL] 정보 보안의 3요소와 Safe-Path 적용 방안

공부한 날짜: 2026-07-01 (수)
참고 자료: 그림으로 배우는 보안 구조

1. 정보 보안의 3요소 (CIA)

기밀성: 인가된 사용자만 볼 수 있게 숨기는 것.

무결성: 데이터가 중간에 조작되지 않게 지키는 것.

가용성: 시스템을 필요할 때 언제든 쓸 수 있게 하는 것.

2. 우리 프로젝트(Safe-Path)에 적용한다면?

기밀성 방어: 어르신의 사생활 데이터를 단말기에서 즉시 파기한다.

무결성 방어: 위험 신호를 조작하지 못하게 암호화하고 다중 승인을 거친다.

가용성 방어: 정전이 되어도 블루투스 메쉬망으로 알림이 울리게 한다.
